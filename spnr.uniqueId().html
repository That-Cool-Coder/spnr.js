spnr.uniqueId()

Reference contents

Syntax:

spnr.uniqueId()

Example:

var id = spnr.uniqueId()

Description:

Use spnr.random() (wrapper of Math.random()) to generate a random, unique number, which is converted into a string. Ids are prefixed with an underscore to allow property naming, etc.
