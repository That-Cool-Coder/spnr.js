Class: MouseWatcher

spnr. MouseWatcher

Class to provide callbacks for mouse events and to allow querying of current mouse state

new MouseWatcher( [elem], scale)

Create a new MouseWatcher

Parameters:
Name Type Argument Default Description
elem Element <optional>
 document

Element to watch. Defaults to the whole document.
scale number 1

Amount to scale the mouse position by. Mainly used in the game engine when a canvas is scaled.
